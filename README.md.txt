# E-commerce Sales Analysis & Dashboard (Excel)

## Project Overview
This project analyzes e-commerce sales data using Microsoft Excel to generate actionable business insights. 
The dataset includes customer, product, store, and sales dimensions.

## Objectives
- Clean and transform raw sales data
- Perform descriptive statistical analysis (mean, median, mode)
- Analyze product-wise and store-wise sales performance
- Understand customer behavior by age group and gender
- Create sales dashboards for decision-making

## Tools Used
- Microsoft Excel

## Key Analysis
- Data cleaning and transformation
- Pivot tables for sales analysis
- Product and store sales dashboards
- Year-wise sales trend analysis
- Inventory and stock-level insights

## Key Insights
- Identified top-performing products and stores
- Analyzed customer segments driving higher sales
- Highlighted sales trends across different years
